The TABLE element has the following attributes:

  1. BORDER, this instructs the browser to display the table with borders around all table cells.
  2. CELLSPACING=num, this allows the amount of space inserted between individual cells to be altered.
  3. CELLPADDING=num allows the amount of space between the border of a cell and the contents of the cells to be altered.
  4. WIDTH=num or percent. It is used to define the desired width of the table
